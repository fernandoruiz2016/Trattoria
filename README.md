# Trattoria Bella - Desarrollo de Interfaces 2
## Instalación

Utiliza la version de [Node.js](https://nodejs.org/) mayor a la v22

Estos son los comandos a utilizar

```sh
git clone https://github.com/fernandoruiz2016/Trattoria
cd react-my-app
npm install
npm run dev
```
